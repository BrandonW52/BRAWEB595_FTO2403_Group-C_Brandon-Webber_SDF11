# Installation

```
# Install dependencies
npm install

# Start the development server with live preview using Browsersync
npm run browsersync

# Compile Tailwind CSS
npm run build

# Compile Tailwind CSS minified
npm run build-minify

```
P.S. node modules is not respecting .gitignotre
